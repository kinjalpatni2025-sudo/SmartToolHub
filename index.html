FAQ

Are my files uploaded to a server?

No. All tools here run in your browser using client-side JavaScript, keeping your files private on your device.

Why is Smarttool Hub fast?

We use optimized, lightweight code and process files locally. No queueing or server round-trips.

Can I use it offline?

Yes. After the page and a tool load once, most features continue to work offline in modern browsers.

Is it free for commercial use?

Yes. Tools are free to use. Please review each tool’s license if you fork or redistribute.
