Important: Use a strong password. If you lose it, the encrypted file cannot be recovered.

Decrypt a file

How this works (short)

- Derive key from password using PBKDF2 (SHA-256) with random salt (200,000 iterations).
- Encrypt file bytes using AES-GCM with a random IV.
- Container format:
  [MAGIC][VERSION(1)][SALT(16)][IV(12)][NAME_LEN(2)][ORIG_NAME][CIPHERTEXT]
- All operations happen in your browser (Web Crypto). This is NOT Adobe PDF password protection.
